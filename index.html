3-Day Plan

From parts on the table to robot in motion.

The exact daily timing can be adjusted, but the learning arc is designed for three focused workshop days.

01

Build the Base

  • Meet the robot kit and core components
  • Assemble the chassis, wheels, motors, and wiring
  • Learn safety, tools, and teamwork habits
02

Code the Brain

  • Connect movement to code commands
  • Use sensors and simple decision logic
  • Debug problems through testing rounds
03

Challenge and Demo

  • Complete a robot mission challenge
  • Improve speed, accuracy, and reliability
  • Present the build to parents and guests

FAQ

Quick answers before families sign up.

Does my child need robotics or coding experience?

No. The workshop is built for beginners and starts from the basics.

What ages is Spark Builders for?

The flyer defines the age range as 11-16. If you want younger or older groups later, the site can add separate cohorts.

Where and when is the workshop?

The current flyer says Casablanca, Summer 2026. Add the exact venue, dates, and daily time once confirmed.

How many students can join?

Each group is limited to 12 students so the workshop stays practical and instructor support stays close.

What should students bring?

Comfortable clothes, curiosity, and water. The workshop materials and tools can be provided on site.
